title: ERDiagram
description: Entity-relationship diagram — titled entity boxes with field rows, joined by cardinality-marked connectors.
---

import PropTable from '../../../components/PropTable.astro';

`entities` are titled boxes with optional `fields` (`{ name, type?, key? }`, where `key` is `'PK'` / `'FK'`); `relationships` join them with orthogonal connectors carrying cardinality markers.

```tsx
import { ERDiagram } from 'goldenchart';

<ERDiagram
  width={520}
  height={340}
  entities={[
    {
      id: 'user',
      label: 'User',
      fields: [
        { name: 'id', type: 'uuid', key: 'PK' },
        { name: 'email', type: 'text' },
      ],
    },
    {
      id: 'order',
      label: 'Order',
      fields: [
        { name: 'id', type: 'uuid', key: 'PK' },
        { name: 'user_id', type: 'uuid', key: 'FK' },
        { name: 'total', type: 'numeric' },
      ],
    },
  ]}
  relationships={[
    { from: 'user', to: 'order', label: 'places', fromCardinality: '1', toCardinality: 'N' },
  ]}
/>;
```
