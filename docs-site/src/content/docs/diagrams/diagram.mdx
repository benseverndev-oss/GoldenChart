`Diagram` is the engine the node-link family is built on: you pass a `layout` engine and it positions nodes, draws edges (curved or orthogonal, with optional arrowheads + knockout labels), and renders group containers. Reach for it when you want a layout the presets don't cover.

```tsx
import { Diagram } from 'goldenchart';
import { flowLayout } from 'goldenchart';

<Diagram
  width={480}
  height={320}
  layout={flowLayout('TB')}
  nodes={[
    { id: 'a', label: 'A' },
    { id: 'b', label: 'B' },
  ]}
  edges={[{ from: 'a', to: 'b' }]}
/>;
```

Available layout engines include `flowLayout(direction, opts)`, `radialLayout()`, and `architectureLayout(direction, opts)`.
