# Bundled fonts

Each vibe preset ships with a matching open-source font, subsetted to Latin
(`U+0020–00FF` plus common punctuation) and embedded as base64 TTF in
`src/assets/fonts.ts`. This lets a vibe's typography render identically in a
browser or a headless rasterizer without any network request or locally
installed font.
